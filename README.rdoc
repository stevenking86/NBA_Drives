== NBA Drives

NBA Drives is a new way to display some of the advanced data about each NBA player's drives.  According to stats.nba.com a drive is "any touch that starts at least 20 feet of the hoop and is dribbled within 10 feet of the hoop and excludes fast breaks".

To Run NBA Drives:
1. Clone this repo.
2. Run rails s
3. Navigate to localhost:3000
